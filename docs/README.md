Obtain API Credentials\n- Log into your S&S Activewear account\n- Navigate to My Account → API Settings\n- Generate or retrieve your API key\n- Your username is your account number\n\n### 2. Test Your Connection\n```bash\ncurl -u \"your_account:your_api_key\" \\\n  \"https://api-ca.ssactivewear.com/v2/styles/\" \\\n  -H \"Accept: application/json\"\n```\n\n### 3. Explore the Catalog\nStart with basic catalog queries:\n```bash\n# Get all categories\ncurl -u \"account:key\" \"https://api-ca.ssactivewear.com/v2/categories/\"\n\n# Get popular styles\ncurl -u \"account:key\" \"https://api-ca.ssactivewear.com/v2/styles/?search=Gildan\"\n\n# Check inventory for a style\ncurl -u \"account:key\" \"https://api-ca.ssactivewear.com/v2/inventory/00760\"\n```\n\n### 4. Implement Error Handling\nAlways implement proper error handling and respect rate limits:\n\n```javascript\nconst rateLimitDelay = (ms) => new Promise(resolve => setTimeout(resolve, ms));\n\nasync function apiCall(endpoint) {\n  try {\n    const response = await fetch(`https://api-ca.ssactivewear.com/v2/${endpoint}`, {\n      headers: {\n        'Authorization': `Basic ${btoa('account:key')}`,\n        'Accept': 'application/json'\n      }\n    });\n    \n    if (response.status === 429) {\n      await rateLimitDelay(60000); // Wait 1 minute\n      return apiCall(endpoint); // Retry\n    }\n    \n    return response.json();\n  } catch (error) {\n    console.error('API Error:', error);\n    throw error;\n  }\n}\n```\n\n## Common Use Cases\n\n### Product
